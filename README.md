# Install POWHEGBOX2

Take the code from the repository:
```
svn checkout --username anonymous --password anonymous svn://powhegbox.mib.infn.it/trunk/POWHEG-BOX-V2
```

Setup environment and LHAPDF_DATA_PATH from cvmfs:
```
source setup.sh
```

Compile:
```
make
```
